Diseño Analítico de un Controlador PI Analógico y por Rediseño Digital

Diseño analítico de un controlador PI analógico y digital para un sistema de primer orden (circuito RC).

planta: Gp(s)=gain/(s+1/tau)

Nota: fs = 30Hz.

Se resuelven en el siguiente repo:

  • [x] Diseño analógico de un controlador PI.
  • [x] Rediseño digital de un controlador PI por Tustín.
  • [x] Comparación entre control PI y PID.

La documentación con mas detalle puede ser encontrada en el directorio paper.

MATLAB

Se identifica el sistema usando la estructura parametrica ARX.

>> iden_arx.m

Hallamos los controladores PI y exportamos los coeficientes obtenidos del controlador digital al directorio /data para su procesamiento en labview.

>> control_pi.m

Hallamos los controladores PID y exportamos los coeficientes obtenidos del controlador digital al directorio /data para su procesamiento en labview.

>> control_pid.m

LABVIEW

Identificación de nuestra plata (Para comparar con el resultado de matlab obtenido en iden_arx.m)

>> arx.vi

Validamos nuestro controlador digital el modelos de nuestra planta identificada.

>> simulado.vi

Validación en tiempo real con nuestra planta experimental.

>> tiempo_real.vi

Advertencia that to run all the labview files. First you need to change the absolute paths to the ones that belong you.

DATA

Data adquirida con una DAQ.

>> data_rc.lvm

Coeficientes generados, representan el modelo de la planta identificada.

>> coef_planta.lvm

Numerador del controlador digital.

>> num_controller.lvm

Denominador del controlador digital.

>> den_controller.lvm

Nota Dependiendo que script ejecutemos:

  • control_pi.m
  • control_pid.m

, obtendremos los coeficientes generados en nuestra /data para nuestro controlador digital.
